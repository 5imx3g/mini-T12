#### 按键功能定义
* 长按操作（5下短音最后1下长音）
  * 主界面，进入设置界面
  * 其他界面，退出至主界面
* 双击操作（2下短音）
  * 主界面，加热或停止状态切换
  * 其他界面，无
* 单击（1下短音）
  * 主界面，无
  * 设置界面，进入二级菜单
  * 二级菜单，切换数值更改选中状态，或确认更改数值
#### 菜单选项
* PID
  * P
  * I
  * D
  * K
* 休眠
  * 休眠时间
  * 休眠温度
* 屏幕
  * 屏幕亮度
  * 屏幕方向
  * 编码器方向
* 电源
  * 基准电压
  * 电影电压
  * 低压报警
  * 过流报警
* PWM（暂无）
  * PWM频率
  * PWM MAX
* 烙铁
  * 烙铁头
  * 冷端补偿
  * 开机加热
  * 恢复出厂设置
